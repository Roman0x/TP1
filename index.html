Développement WEB

Définitions

La page web est obtenue par interprétation d'un code HTML envoyé par le serveur en réponse à la requête qui lui est envoyée (adresse en http://www.etc). C'est le navigateur qui interprète le code.

Il existe deux autres sortes de balises :

  1. Les balises les plus courantes dans le cadre su code qui se termine par une balise dite fermante. Exemple : <balise> contenu HTML </balise>
  2. Les balises dites orphelines qui n'encadrent aucun code et qui se ferment toutes seules. Exemple : <br/> <hr/> <img/> <input/> etc ...

Balises HTML
  • main (Balise principale)
    • <header> - Defines a header for a document or a section
    • <nav> - Defines a set of navigation links
    • <section> - Defines a section in a document
    • <article> - Defines an independent, self-contained content
    • <aside> - Defines content aside from the content (like a sidebar)
    • <footer> - Defines a footer for a document or a section
    • <details> - Defines additional details that the user can open and close on demand
    • <summary> - Defines a heading for the <details> element
